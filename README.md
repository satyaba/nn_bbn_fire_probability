# NN-Enhanced BBN Fire Probability

To calculate fire probability given study area. The repository is the minimal implementation to do calculation. The script doesn't contain any training step and solely to do forward prediction.

The minimum required dataset and model to run the algorithm also available in this repository.

The script designed to run on Google Colab.

> WARNING :
> Even though the script has been tested on author's machine to run using GPU runtime in the Google Colab, there are possibility of error due to incompatibility of library related to CUDA, specifically numba. If this happened, consult to https://github.com/googlecolab and https://github.com/NVIDIA/numba-cuda or run the CPU Runtime (may take hours to complete)
